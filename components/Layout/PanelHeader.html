PanelHeader

Практически всегда содержимое панели должно начинаться с Separator, поэтому он рисуется в PanelHeader по-умолчанию.

Важно

Правая часть шапки будет скрыта, если в ConfigProvider передан webviewType={WebviewType.VKAPPS}.

Свойства

Name Type Default Description
separator boolean true
transparent boolean false
shadow boolean false
visor boolean true Если `false`, то шапка будет нулевой высоты и контент панели "залезет" под неё
fixed Если `false`, то шапка будет в потоке. По умолчанию `true`, но если платформа vkcom, то по умолчанию `false`.

Этот компонент использует $$restProps

Все прочие свойства будут переданы базовому элементу div
