UsersStack

Используется для отображения аватарок пользователей с каким-то сопроводительным текстом.

У компонента есть вертикальный режим – когда текст располагается под аватарками. В таком режиме рекомендуется использовать размер m.

Свойства

Name Type Default Description
photos [] [] Массив ссылок на фотографии
size string 's' Размер аватарок
layout string 'horizontal' Вертикальный режим рекомендуется использовать с размером `m`
visibleCount number 3 Количество аватарок, которые будут показаны. Если в массиве `photos` больше элементов и используется размер `m`, то будет показано количество остальных элементов

Этот компонент использует $$restProps

Все прочие свойства будут переданы базовому элементу div
