Свойства

Name Type Default Description
mode string 'tint' Тип баннера.
size string 's'
asideMode Тип действия в правой части баннера. - `dismiss` – отображается иконка крестика, при клике на неё сработает свойство `onDismiss`. - `expand` – отображается иконка шеврона, которая подразумевает, что при клике на баннер можно куда-то перейти.
dismissLabel string 'Скрыть' `aria-label` для кнопки при `asideMode="dismiss". Необходим, чтобы кнопка была доступной.
before string '' Содержимое, отображаемое в левой части баннера.
header string '' Заголовок.
При использовании этого свойства рекомендуется не указывать `text`.
subheader string '' Подзаголовок.
При использовании этого свойства рекомендуется не указывать `text`.
text string '' Текст баннера.
Это свойство следует использовать без указания `header` и `subheader`.
imageTheme string 'dark' При использовании `mode="image"`. - `light` – в качестве фона используется светлое изображение, цвет текста в баннере будет тёмным. - `dark` – в качестве фона используется тёмное изображение, цвет текста будет светлым.
background string '' При использовании `mode="image"`. Элемент, который нужно стилизовать цветом и/или фоном. Этот элемент будет растянут на 100% ширины и высоты баннера.
actions string '' Кнопки, отображаемые в баннере. - В режиме `tint` или в `image` со светлым фоном рекомендуется использовать только `

Этот компонент использует $$restProps

Все прочие свойства будут переданы базовому элементу section
