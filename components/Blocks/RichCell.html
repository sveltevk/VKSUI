Свойства

Name Type Default Description
text string '' Контейнер для текста под `children`.
caption string '' Контейнер для текста под `text`.
bottom string '' Контейнер для контента под `caption`. Например ``
after string '' Иконка 28 или текст
multiline boolean false

Этот компонент использует $$restProps

Все прочие свойства будут переданы базовому элементу Tappable
