Свойства

Name Type Default Description
component
href
indicator string '' Контейнер для текста справа от основного текста.
description string '' Контейнер для текста под основным текстом.
disabled boolean false Убирает анимацию нажатия
expandable boolean false В iOS добавляет chevron справа. Передавать `true`, если предполагается переход при клике по ячейке.
multiline boolean false

Этот компонент использует $$restProps

Все прочие свойства будут переданы базовому элементу Tappable
