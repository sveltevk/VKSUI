Кастомизация

Есть возможность переопределить жирность.

Пример реализации

platform:
appearance:
Caption 1 weight 1
Caption 1 weight 2
Caption 1 weight 3

Код

<div style:padding="20px">
	<Caption level="1" weight="1" style="margin-bottom:16px">Caption 1 weight 1</Caption>
	<Caption level="1" weight="2" style="margin-bottom:16px">Caption 1 weight 2</Caption>
	<Caption level="1" weight="3">Caption 1 weight 3</Caption>
</div>

Свойства

Name Type Default Description
component string 'div'
weight
level string '1'
caps boolean false

Этот компонент использует $$restProps

Все прочие свойства будут переданы базовому элементу span
