ConfigProvider

Компонент для пробрасывания конфигурации приложению. Помимо прочего, отвечает за установку темы и платформы.

Свойства

Name Type Default Description
scheme string 'bright_light' Цветовая схема приложения
isWebView boolean false Подсказывает приложению, обёрнутому в `ConfigProvider`, где открыто приложение: внутри webview или в мобильном браузере
webviewType Тип вебвью.
В случае `WebviewType.VKAPPS` интерфейс будет адаптирован для отображения в вебвью Mini Apps (системные контролы в правой части шапки)
app string '' Тип приложения
appearance string 'light' Тип цветовой схемы – `light` или `dark`
transitionMotionEnabled boolean true Включена ли анимация переходов между экранами в `Root` и `View`
platform Платформа
hasNewTokens boolean false
hasMouse
contentWindow
contentDocument
sizeX
